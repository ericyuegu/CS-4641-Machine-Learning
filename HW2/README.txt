The code for this assignment can be found in three files, all within /ABAGAIL/src/opt/test/:

CancerTest.java
MaxKColoringTest.java
FourPeaksTest.java

To get started with ABAGAIL in command line, follow the instructions here (installing Java 8 SDK and Ant):
https://github.com/pushkar/ABAGAIL/blob/master/faq.md#user-content-how-to-get-started-with-abagail

To compile and run:
cd ABAGAIL
ant
java -cp ABAGAIL.jar opt.test.CancerTest
java -cp ABAGAIL.jar opt.test.MaxKColoringTest
java -cp ABAGAIL.jar opt.test.FourPeaksTest
